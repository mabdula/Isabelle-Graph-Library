# Isabelle-Graph-Library

A trimmed version of the graph library, with results associated with the manuscript "A Formal Correctness Proof of Edmonds's Blossom Shrinking Algorithm". There are three relevant sessions for the manuscript:

 - Undirected_Set_Graphs containing a set-based simple representation of unirected graphs.
 - Berge_Lemma containing a formalisation of Berge's Lemma.
 - Blossom containing the formalisation of the correctness proof of Edmonds' blossom shrinking algorithm.

All theories are compatible with Isabelle2024.
